# punchie

A fun way to track habits and small goals. Create punch cards, stamp a slot each time you complete your goal, and treat yourself when the card is full.

Cards are stored in your browser's localStorage.

## Features

- Create and customize punch cards with a title and reward
- Pick a color, slot shape, and stamp icon
- Punch slots as you make progress
- Download your card as a PNG to print and use physically
- Cards split into active and completed

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Base UI
- Phosphor Icons

## Getting started

```bash
npm install
npm run dev
```
